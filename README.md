## Blockend
A simple experiment of creating a backend that connects and syncs data from a blockchain smart contract and saves it inside mongodb

### Technology Used
- NodeJS
- TypeScript
- RabbitMQ
- Mongoose
- MongoDB
- ExpressJS
- SocketIO
- Docker

### Information
- Graphql Endpoint and Playground: http://localhost:3001/v1/graphql/
